# NSFWTrack — 项目进度与开发规划

> NSFWTrack 是本地单用户媒体记录器 / 收藏管理器。  
> 当前开发边界：本地管理、本地数据维护、手动确认操作、SQLite、FastAPI、Jinja2、轻量原生 JavaScript、Docker Compose。
> Phase 3-A1 允许保存用户提供的 URL，并解析本地书签 HTML / 纯文本 URL 清单；仍禁止请求外部网页、爬虫、远程图片、站点 adapter、自动同步、推荐、AI、多用户或云同步。
