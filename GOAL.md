# 当前目标：Phase 3-A5 — 媒体库检索与分页

## 目标

让本地媒体库在文件较多时仍能快速浏览和定位图片。

## 任务

- 媒体列表增加文件名和路径搜索
- 支持全部、可用、损坏、已使用、未使用筛选
- 支持文件名和大小排序
- 媒体列表每页 20 条
- 翻页时保留搜索、筛选及 A3/A4 页码
- 空结果和非法参数安全降级
- 同步中英文、文档和测试

## 边界

- 不修改媒体文件或关联
- 不请求外部网络，不使用 AI
- 不新增表，不修改 Schema 2、依赖或版本
- 不改变 A3/A4 候选生成逻辑
- 不部署到 N100

## 完成标准

- 搜索、筛选、排序和分页结果稳定
- GET 操作零写入
- pytest、pip check、Docker 和 Actions 通过
- 提交推送，不创建 Release
