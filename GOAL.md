# 当前目标：Phase 3-A2 — 本地媒体库

请先读取 `RULE.md`、`PLAN.md` 和现有本地媒体代码。

## 目标

支持从 WebUI 上传和管理本地图片，并设置为条目封面或创作者头像。

## 任务

- 扫描 `data/media`，展示图片及引用状态
- 支持单图和多图上传
- 校验大小、数量、扩展名、MIME 和文件头
- 使用 SHA-256 避免重复保存
- 支持设置、替换和清除封面或头像
- 文件缺失、损坏或非法时安全处理
- 同步中英文、文档和测试

## 边界

- 不跟随符号链接，不越出媒体目录
- 不允许 SVG、HTML 或伪装文件
- 不物理删除媒体文件
- 不新增表，不修改 Schema 2、依赖或版本
- 不请求外部 URL，不获取远程图片
- 保持现有 Docker 安全配置
- 不部署到 N100

## 完成标准

- 上传、扫描、去重和关联功能正常
- 清除关联不删除文件
- Docker 重建后图片和关联保持
- pytest、pip check、Docker 和 Actions 通过
- 提交并推送，不创建 Release
