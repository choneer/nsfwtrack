# 当前目标：Phase 3-A3 — 本地媒体批量配对

## 目标

根据本地图片文件名，为条目封面和创作者头像生成候选配对，并由用户预览确认。

## 任务

- 对未使用媒体、无封面条目和无头像创作者生成候选
- 支持精确名称、规范化名称和约定后缀匹配
- 显示匹配依据及置信等级
- 歧义候选必须标记冲突
- 支持单项确认和当前页批量确认
- 已有关联默认不覆盖
- 同步中英文、文档和测试

## 边界

- 不自动应用候选
- 不使用 AI 或内容识别
- 不请求外部 URL
- 不新增表，不修改 Schema 2、依赖或版本
- 不删除、移动或重命名媒体文件
- 不部署到 N100

## 完成标准

- 候选生成稳定且可解释
- 歧义和已有关系不会被错误覆盖
- 所有写操作使用 POST 和确认
- pytest、pip check、Docker 和 Actions 通过
- 提交并推送，不创建 Release
