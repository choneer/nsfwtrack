# GOAL.md

# 当前目标：Phase 3-A4 — 未匹配媒体快速建档

## 目标

根据未使用的本地图片生成新条目候选，确认后创建条目并设置封面。

## 任务

- 从有效、未使用且未匹配的媒体生成候选
- 根据文件名生成标题，`.cover` 后缀自动移除
- `.avatar` 文件不生成条目
- 允许确认前修改候选标题
- 已有同名或规范化同名条目标记冲突
- 同批次标题冲突必须拒绝
- 支持单项和当前页批量确认
- 写入前重新扫描并整批事务提交
- 同步中英文、文档和测试

## 边界

- 不自动创建条目
- 不覆盖已有条目或媒体关联
- 不移动、重命名或删除媒体
- 不请求外部网络，不使用 AI
- 不新增表，不修改 Schema 2、依赖或版本
- 不部署到 N100

## 完成标准

- 候选生成和冲突判断稳定
- 创建后封面关联正确
- 陈旧或伪造提交被拒绝
- 批量失败整批回滚
- pytest、pip check、Docker 和 Actions 通过
- 提交推送，不创建 Release
